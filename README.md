# tic-tac-toe

Крестики-нолики:
режим 1 на 1
режим игры против компьютера

Установка:

mkdir ticTacToe /n
cd ./ticTacToe /n
git clone https://github.com/VKNS/tic-tac-toe.git /n 
cd ./tic-tac-toe/ /n
yarn (установка зависимостей довольно долго идет)/n

Запуск:

yarn run start
переходим http://localhost:3000/
