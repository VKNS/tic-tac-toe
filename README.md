# tic-tac-toe

Крестики-нолики:

  режим 1 на 1
  
  режим игры против компьютера

Установка:

  mkdir ticTacToe 
  
  cd ./ticTacToe 
  
  git clone https://github.com/VKNS/tic-tac-toe.git 
  
  cd ./tic-tac-toe/ 
  
  yarn (установка зависимостей довольно долго идет)

Запуск:

  yarn run start
  
  переходим http://localhost:3000/
